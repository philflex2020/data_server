Chiller Controller — Heltec LoRa32

Liquid cooling chiller simulator/controller with auto temp control loop and WebSocket interface

ws :81 Heltec LoRa32 V2/V3 LILYGO T-Display
Open Control UI Download chiller_control.ino

Overview

Simulates or controls a liquid cooling chiller. Drives compressor speed and fan speed via DAC, controls pump and compressor contactors via GPIO, and reads inlet/outlet temperatures from 4–20 mA ADC inputs. Runs an automatic temperature control loop to maintain a configurable outlet setpoint.

Hardware

GPIO26 / DAC2Fan speed reference0–3.3 V = 0–100%

WebSocket Commands (JSON, port 81)

{"cmd": "get_status"}
{"cmd": "set_setpoint",   "value": 18.0}       // outlet temp °C
{"cmd": "set_compressor", "value": 75.0}       // % (overrides auto)
{"cmd": "set_fan",        "value": 60.0}       // %
{"cmd": "set_mode",       "mode": "cooling"}   // cooling | standby | manual
{"cmd": "set_pump",       "state": true}
{"cmd": "clear_fault"}
{"cmd": "ping",           "t": 0}
