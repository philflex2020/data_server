Load Controller — Heltec LoRa32

ESP32 DAC load controller with WebSocket control, OLED display, OTA, and waveform profiles

ws :81 Heltec LoRa32 V2/V3 ESP32-S3
Open Control UI Download load_controller_lora32.ino Download fast_load_control.ino

Overview

Runs on a Heltec LoRa32 V2 (ESP32) or V3 (ESP32-S3). Serves a WebSocket on port 81 for browser control. Drives two DAC channels (voltage + current) and executes ramp/profile sequences. The fast_load_control variant uses I2S DMA at 5000 Hz for smooth 200 µs steps instead of the standard ~10 ms steps.

WebSocket Commands (JSON, port 81)

{"cmd": "set_voltage",    "value": 48.0}
{"cmd": "set_current",    "value": 100.0}
{"cmd": "set_dac",        "channel": 0, "value": 2048}
{"cmd": "set_led",        "state": "blink"}
{"cmd": "get_status"}
{"cmd": "load_profile",   "profile": {...}}
{"cmd": "start_profile",  "channel": "voltage"|"current"|"both"}
{"cmd": "stop_profile",   "channel": "voltage"|"current"|"both"}
{"cmd": "set_display",    "channel": "voltage"|"current"}
