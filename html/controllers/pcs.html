PCS Controller — Heltec LoRa32

Bidirectional Power Conversion System (AC↔DC) simulator/controller with charge/discharge/standby modes

ws :81 Heltec LoRa32 V2/V3 LILYGO T-Display
Open Control UI Download pcs_control.ino

Overview

Simulates or controls a bidirectional Power Conversion System. Drives DC voltage and current limit references via DAC, controls grid and DC bus contactors via GPIO, and supports charge, discharge, standby, and fault modes. Accepts a signed power setpoint (positive = charge, negative = discharge).

Hardware

PinFunctionNotes
GPIO25 / DAC1DC voltage reference0–3.3 V = 0–800 V DC
GPIO26 / DAC2Current limit reference0–3.3 V = 0–500 A

WebSocket Commands (JSON, port 81)

{"cmd": "get_status"}
{"cmd": "set_mode",          "mode": "charge"}     // charge | discharge | standby | fault
{"cmd": "set_dc_voltage",    "value": 750.0}        // V
{"cmd": "set_current_limit", "value": 200.0}        // A
{"cmd": "set_soc_target",    "value": 80.0}         // %
{"cmd": "set_power",         "value": 50000.0}      // W (+ charge, - discharge)
{"cmd": "clear_fault"}
{"cmd": "ping",              "t": 0}
