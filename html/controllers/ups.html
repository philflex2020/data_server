UPS Controller — Heltec LoRa32

Uninterruptible Power Supply simulator/controller with normal, bypass, battery, and test modes

ws :81 Heltec LoRa32 V2/V3 LILYGO T-Display
Open Control UI Download ups_control.ino

Overview

Simulates or controls a UPS. Drives output voltage reference and battery current sense via DAC, controls mains bypass contactor and inverter enable via GPIO. Supports normal, bypass, battery, test, and fault modes with a battery test command.

Hardware

PinFunctionNotes
GPIO25 / DAC1Output voltage reference0–3.3 V = 0–240 V AC
GPIO26 / DAC2Battery current sense0–3.3 V = 0–100 A
GPIO OUT[0]Mains contactor (bypass)Digital output
GPIO OUT[1]Inverter enableDigital output
GPIO OUT[2]Alarm relayDigital output

WebSocket Commands (JSON, port 81)

{"cmd": "get_status"}
{"cmd": "set_mode",           "mode": "normal"}  // normal | bypass | battery | test | fault
{"cmd": "battery_test"}                           // initiate battery test
{"cmd": "set_output_voltage", "value": 230.0}    // V AC
{"cmd": "set_load",           "value": 60.0}     // % (sim only)
{"cmd": "clear_fault"}
{"cmd": "ping",               "t": 0}
